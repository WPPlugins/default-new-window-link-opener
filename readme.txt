Plugin for wordpress to enable, in the popup insert/edit link, the check to open a link in a new window by default.

== Description ==

The default functionality for a link is to be in the same page, if you want the insert/edit link to be opened by default in a new page, you need to create the link and then enable the check box. This can be quite anoying when you always want to open a new window.

This plugin enable by default the checkbox to open a link in a new window everytime you insert/edit link.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/default-new-window-link-opener` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
